# AI 大模型：它凭什么这么强？

> 你每天都在用 AI——写代码、查资料、翻译、生成图片。
> 但你有没有想过：它凭什么能做这些事？它是怎么"变聪明"的？
> 这篇是整个系列的起点。

---

## 它能做什么？

先看几个你可能已经习以为常、但在 5 年前完全不可能的事情：

```
日常对话
──────────────────────────────────────
  你: "帮我解释一下数据库索引的原理"
  AI: 从 B+ 树讲到磁盘 I/O，举例说明，5 分钟讲清楚
  → 以前要翻半小时文档才能拼出来的东西

写代码
──────────────────────────────────────
  你: "用 Go 写一个并发安全的 LRU Cache"
  AI: 直接给出完整实现，带注释、带测试用例
  → 以前 Stack Overflow 搜半天，还要自己拼凑

长文写作
──────────────────────────────────────
  你: "帮我写一份微服务架构迁移方案"
  AI: 输出背景分析、方案对比、实施步骤、风险评估
  → 以前要写几天的文档

多步推理
──────────────────────────────────────
  你: "一个水池，进水管 3 小时灌满，排水管 5 小时排空，
      同时打开，多久灌满？"
  AI: 列方程、解方程、给出答案 7.5 小时
  → 不是查表，是真的在"算"

多语言翻译
──────────────────────────────────────
  你: 贴一段日文技术文档
  AI: 准确翻译，连专业术语都不出错
```

这些能力在 2020 年之前是**不存在的**。

## 它为什么这么聪明？

答案出乎意料的简单——它做的唯一一件事就是**"预测下一个词"**。

```
你输入: "法国的首都是"

AI 内部:
  → 根据 700 亿个参数做运算
  → 输出概率: "巴黎" 92%,  "马赛" 3%,  "里昂" 2%, ...
  → 选概率最高的: "巴黎"

继续:
  → "它" → "位于" → "塞纳河畔" → ...

就这样一个词一个词地"接龙"出完整的回答。
```

**就"接龙"能写代码？能推理？** 能。关键在于三件事同时到位了：

```
① "大脑"足够大
   700 亿个参数 → 能存储几亿条"知识碎片"
   → 记住了代码的模式、语言的结构、推理的步骤

② 读的"书"足够多
   训练数据: 15 万亿 token ≈ 1100 万本书
   → 读过几百万份代码、几千万篇文章、几百万道数学题
   → 模型读的书比任何人一辈子能读的都多几万倍

③ 理解方式足够强
   不是逐字阅读（读到后面忘了前面）
   → 而是一次看到所有输入，能发现任意两个词之间的关系
   → 就像把整篇文章摊在桌上看，而不是用吸管一个字一个字吸
```

当这三个条件同时满足，简单的"预测下一个词"就**涌现**出了看起来像"智能"的行为——能写代码、能推理、能翻译。这是大模型最反直觉的地方。

## 为什么是最近几年才变聪明？

这三个条件不是天上掉下来的——它们在 2017-2020 年间同时到位：

### 架构突破：从逐字阅读到全局理解（2017）

```
以前 (RNN): 逐词阅读，每次只看一个词
  → 读到第 1000 个字时，已经忘了第 1 个字
  → 无法理解远距离的关联

现在 (Transformer): 一次看到所有输入
  → "法国"和"首都"隔了多远都能关联上
  → 而且天然适合 GPU 并行 → 可以堆到几千亿参数
```

### 算力爆炸：GPU 从游戏卡变成 AI 引擎

```
2012: 2 块 GPU,        训 6000 万参数
2020: 几千块 V100,      训 1750 亿参数
2024: 16000 块 H100,    训 4050 亿参数

GPU 单卡算力 12 年增长约 1000 倍
集群规模从 2 卡 → 16000 卡
→ 总算力增长了约 800 万倍
```

### 数据洪流：互联网成了免费教材

```
2018: 几 GB 数据
2020: 570 GB，约 3000 亿 token
2024: 15 万亿 token ≈ 1100 万本书

数据来源: 网页、书籍、论文、代码(GitHub)、百科、论坛...
```

**缺一不可：**
- 只有架构没有算力 → 堆不到几千亿参数
- 只有算力没有数据 → 学不到丰富的知识
- 只有数据没有架构 → 根本处理不了这么大规模

### 进化时间线

```
2012  AlexNet ─────── 6000 万参数，只能认猫认狗
│
2017  Transformer ─── ⚡ 架构突破，理解方式质变
│
2018  GPT-1 ───────── 1 亿参数，开始用"预测下一个词"来学习
│
2019  GPT-2 ───────── 15 亿参数，能写流畅的文本了
│
2020  GPT-3 ───────── 1750 亿参数 → 涌现！突然会写代码、做推理
│
2023  GPT-4 / Llama 2 ── AI 进入主流，万卡训练成为常态
│
2024  Llama 3 / DeepSeek-V3 ── 开源追平闭源，训练成本数千万美元
│
2025  万亿参数时代 ── 你正在用的就是
```

**从认不出猫，到能写论文——只用了 12 年。**
